Mindful Spending Journal
Firestore rules need updating — to read all user journals, add this rule in the Firebase Console under Firestore → Rules: 
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /journals/{userId} {
      allow read, write: if request.auth.uid == userId;
      allow read: if request.auth.token.email == 'sparakala@gmail.com';
    }
    match /_meta/{doc} {
      allow read, write: if request.auth != null;
    }
  }
}

Google Analytics Data Connect to pull live metrics from GA4
One-time setup required:
  1. Go to Google Cloud Console and enable the Google Analytics Data API v1
  2. Under Credentials, find your OAuth 2.0 Web Client ID and paste it as GOOGLE_CLIENT_ID in this file
  3. Add https://www.mindfulmundane.com to Authorized JavaScript Origins on that client
  4. Find your GA4 Property ID in GA4 → Admin → Property Settings and paste it as GA4_PROPERTY_ID
